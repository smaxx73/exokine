# exokine

Exercices de kiné

Ouvrez `index.html` dans votre navigateur pour accéder à la page d'accueil et choisir une séance. Les fichiers décrivant les séances se trouvent désormais dans le dossier `sessions`. Une fois la séance choisie, vous serez redirigé vers `player.html` qui chargera le fichier JSON correspondant.
